BlockDownloadEntity

Package private.Represents metadata for a block download.There is one-to-many relationship between aBlobDownloadEntityand a set ofBlockDownloadEntity.EachBlockDownloadEntityin the set describes a chunk of the file referenced by theBlobDownloadEntityThe Data Access Object typeDownloadDaoexposes DB store and read methods on this model.
